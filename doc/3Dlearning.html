关于系统环境配置部分 see dl-env.html

在你的tensorflow虚拟空间中开始安装必要的三维显示、处理、计算等工具。

安装 pycollada，三维模型处理、读写软件
apt-get install python-lxml python-numpy python-dateutil
easy_install pycollada

安装 PyMCubes，曲面提取工具
pip install --upgrade PyMCubes

安装 OpenCV-Python ，图像读写等
pip install opencv-python，主要模块
pip install opencv-contrib-python，包含主模块和contrib模块
可能很慢，比较快的是去https://pypi.org/project/opencv-python/#files直接下载下来whl文件进行安装pip install x.whl
